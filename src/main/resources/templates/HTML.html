Html Inception

HTML stands for HyperText Markup Language. Web browsers use HTML to turn Text and information into fully visible web pages that we are used to seeing. It is composed of pieces of code called "elements" which take pieces of text and format them into More aesthetic forms. This can include plain text (Like this paragraph itself), or links to websites and images(like the example below). These elements are commonly referred to as tags, which are responsible for distinct functions such as:

  • Heading tags(Like (< h3 >) used for the title in this post)
  • "P" Tags (< p > like the paragraph above)
  • List tags (Like < ul > for this very list)
  • image tags(< img > for displaying the example below)
