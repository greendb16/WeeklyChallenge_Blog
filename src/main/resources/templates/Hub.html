The "Hub" in GitHub

July 11, 2019

GitHub is a web-based Git Repository hosting service that offers all of the distributed revision control and source code management functionality of Git.At the heart of GitHub Git. Git is responsible for everything GitHub-related that happens locally on your computer.Git is a command-line tool, but the center around which all things involving Git revolve is the Hub. it is where developers store their projects and are able to network.

Here are somethings you can do on GitHub...

  • Repository

    • A repository is a location where all the files for a particular project are stored. each new projects has its own repository.

  • Forking a Repository

    • Creating a fork is producing a personal copy of someone's project. forking repository allows you to freely experiment with changes without affecting the original project.

  • Pull Requests

    • Pull requests let you tell others about changes you've pushed to a GitHub repository. Once a pull request is sent, interested parties can review the changes made.
